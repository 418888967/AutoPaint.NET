## What's AutomaticDrawing?
An automatic drawing program, it can control the mouse to draw beautiful images on the screen.

![Preview](Documentation/Image/Preview.png)

## Features

* Load Image
    *  Local files
    *  Screenshot
* Image Processing
    *  Thresholding
    *  Contouring
    *  Hollowing
    *  Clustering
* Extras
    *  Add a signature
    *  Preview Drawing
    *  One-click Drawing
