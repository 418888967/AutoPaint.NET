## What's AutomaticDrawing?
An automatic drawing program, it can control the mouse to draw beautiful images on the screen.
##Featrues

* Load Image
    *  Local files
    *  Screenshot
* Image Processing
    *  Thresholding
    *  Contouring
    *  Hollowing
* Extras
    *  Add a signature
    *  Preview Drawing
    *  One-click Drawing
